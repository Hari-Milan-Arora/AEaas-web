# AEaaS-Web

## Vercel Deployment

1. Import this repository into Vercel.
2. Set the project's Root Directory to `AEaaS-Web`.
3. Keep the framework preset as `Next.js`.
4. Use the default install and build commands, or set them explicitly to `npm install` and `npm run build`.

## Local Verification

- Run `npm run lint` inside `AEaaS-Web`.
- Run `npm run build` inside `AEaaS-Web`.
- If Windows shows mixed-case path warnings, open the repo using the same casing Git reports for the repo root so React is bundled only once.
